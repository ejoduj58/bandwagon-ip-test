## DC3 CN2 是什么，为什么它消失了

**DC3 CN2（编号 USCA_3）** 是搬瓦工位于美国洛杉矶的一个机房，托管在 QuadraNet（QNET）的设施里。它曾经走的是 CN2 GT 线路——通俗来说就是"半程 CN2"，出境段走 CN2 专线，进入国内后部分路段回落到 163 骨干网。比普通直连强一些，比顶级的 CN2 GIA 便宜不少。那时候它是很多人的入门首选。

但这已经是过去的事了。

实测回程路由显示，DC3 现在三网全是普通直连：电信走 163 骨干网，联通走 AS4837，移动走 CMI。CN2 属性已经不存在了。说白了，就算这个机房还能用，它现在和 DC2、DC4、DC8 这些普通线路机房没有任何区别——价格上也不会更便宜。

原因大概率是 QuadraNet 这几年网络质量滑坡，搬瓦工最终决定直接切掉。

结论很清楚了。
